This is:

Who Am I Up Against

A simple Windows app that lets you see the ranks, recent games and other info of players in a live Valorant match while you're still playing it.

Frequently Asked Questions

Can I get banned using this?

No, if Riot doesn't want you using it, they will contact me first.

Doesn't this break Riot's TOS?

On your side, no. On my side, also no. This app uses the ingame API which has no rules.

How is this app even useful?

The main intention is to settle the debate of if they are a smurf or not. Some of the other features like the skin viewer are useful to worthlessly beg for skins, etc.
